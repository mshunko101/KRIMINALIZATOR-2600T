# KRIMINALIZATOR-2600T
Проект исследования многомерных выбросов промышленного оборудования.

# Задача - найти способ минимизации воздействия психо-техногенных выбросов.

Скрипт работает в паре с сенсором, сенсор не стандартизирован, не сертифицирован, не имеет аналогов в Мире.

# Описание сенсора.

Сенсор собран полностью вручную из доступных компонентов:

*   **Основа:** небольшая коробочка.
*   **Активный материал:** графитовые стержни из батареек (дают переменное сопротивление).
*   **Источник энергии:** 10 элементов AAA + 2 элемента A.
*   **Ключевой элемент:** магниты внутри корпуса 3 шт.
*   **4 медные монеты по 5 коп.:** монеты примотаны изолентой, своего рода клещи для снятия напряжения.

**Как это работает:**  
При изменении энергии в системе атомы графита начинают «выбиваться» и под действием магнитного поля концентрируются у полюсов. Это вызывает крайне малые, но измеримые колебания тока. Изначально сигнал пытались усиливать транзистором, но его амплитуды не хватало. Решение нашлось в использовании аналоговых пинов Arduino (в проекте — Vidor 4000), способных фиксировать малейшие колебания напряжения.

Схема подключения:

- Пин №1 сенсора , подключается к источнику напряжения 5Вольт платы Vidor 4000.
- Пин №2 сенсора , подключается к маленькой мантажной плате.
- После Пин №2 сенсора, подключаем ввод пина A2.
- После ввода пина A2, ставим небольшой резистор и замыкаем на  Землю(GND).


# Что показали исследования.

Буквально они могут отобразить, т.е кластеризировать данные поулчаемые из сенсора, обычно выделяется 7-11 кластеров (1), если выпить алкоголя то внезапно кластеров становится 4-6 но не на долго. Под кластерами я понимаю выделеные зоны многомерных выбросов.

# Способ минимизации количества кластеров (n-dim) пока не найден.

Исследования приводят к тому что число кластеров начинает увеличиваться при попытке их минимизировать, пока минимизация заметна только на алкоголе. Максимизация приводит к тому что мозг начинают бамбордировать грубые и тупые мысли, мыслительный процесс затармаживается, по этому проект и назван КРИМИНАЛИЗАТОР-2600Т немного иронично конечно же.
